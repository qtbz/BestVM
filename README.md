## 机房和线路怎么选？

**香港机房**

香港有三个系列：BGP、BGP-Standard和BGP-Lite。

BGP系列走的是CMI直连，也就是中国移动国际线路，三网回国都走这条。延迟低，速度稳定，适合对回国速度有要求的用户。价格相对也高一些。

**日本机房**

日本有三个系列：BGP-B、BGP和BGP-Lite。

BGP-B主打移动优化，线路包含GSL、Lumen、SoftBank。如果你是移动用户，或者主要服务移动宽带的访客，这个系列体验会比较好。联通走的是BBTEC单线，速度也还行。

BGP系列三网走IIJ线路，还包含GSL，提供流媒体解锁服务。IIJ是日本老牌运营商，网络质量相对稳定。这个系列的特色是专门提供流媒体解锁，看Netflix、Disney+没问题。

BGP-Lite走ntt+cogent+pccwg+he，纯国际互联网络，提供IPv4+IPv6。跟香港Lite系列一样，不保证回国速率，但国际访问体验优秀。

**台湾机房**

台湾BGP-Lite包含TPIX、STUIX、Cogent、HE等线路。到国内延迟基本在50-80ms区间，速度表现不错。这个机房还提供台湾家宽动态IP的特殊产品，适合需要真实家庭网络环境的场景。

**新加坡机房**

新加坡BGP-Lite线路包括CDN77、HE、cogent、EIE、SGIX，提供IPv6流媒体解锁服务。注意，这个产品也是无回国优化的。新加坡机房的优势在于东南亚地区访问体验好，带宽充裕，适合面向国际用户的业务。

## 流媒体解锁情况

BestVM的部分产品提供流媒体解锁服务，主要有两种方式：

一种是提供DNS解锁，商家会在后台设置特定的DNS服务器，把Netflix、Disney+等流媒体的请求解析到他们的代理服务器，再转发给真正的流媒体服务器。这样流媒体看到的是商家的IP，你就能解锁了。日本BGP系列和香港BGP-Standard系列就是用这种方式。

另一种是直接提供原生IP，配合IPv6地址实现解锁。台湾、新加坡、日本的Lite系列都提供IPv6，很多流媒体对IPv6的检测没那么严格，成功率也挺高的。

根据用户的反馈，BestVM的日本和新加坡线路解锁Netflix、Disney+、TikTok这些都没压力。香港线路也能解锁主流流媒体，不过偶尔会有波动。台湾线路对台湾本地的流媒体平台支持更好。

需要注意的是，Lite系列虽然提供IPv6解锁，但因为是纯国际线路，回国速度不保证。如果你主要在国内使用，还是推荐选带回国优化的BGP或BGP-B系列。
